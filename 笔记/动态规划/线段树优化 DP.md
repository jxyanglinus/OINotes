# 概念
在 DP 过程中用线段树对 DP 的转移进行优化，就被叫做线段树优化 DP。~~废话~~

例如，题目需要先前状态的最小值来转移，不加优化时，转移一次需要 $O(n)$；如果我们用线段树来维护最小值，那么找一步的复杂度可以降低到 $O(\log n)$。
